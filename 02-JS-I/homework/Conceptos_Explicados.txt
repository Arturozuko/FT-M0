Variables = Sabemos que el tiempo pasa segundo a segundo, cuando tu quieres ver que hora es, miras un reloj, el tiempo entonces es variable y para verlo lo ponemos en un reloj, aca es lo mismo, en lugar de un reloj usas una palabra y en lugar de almacenar tiempo, almacenas cualquier cosa que sea variable
Strings = todo aquello que va entre comilla simple ' ', no importa si para ti es un numero mientras este en comillas simples para la computadora es un string, y como quieres comunicarte con la computadora entonces tienes que hablar su idioma mas no la computadora el tuyo
Funciones (argumentos, return) = imagina que enfermaste, y tienes que tomar las mismas 3 pastillas cada 8 horas, tu puedes agarrar cada vez cada uno de los frasquitos, abrirlos, sacar una pildora, juntar las 3 pildoras y espues tomarlas, ooooooooooooo puedes tomar un pastillero que viene seccionado por cajitas y de una vez pones las 3 pastillas en cada pastillero, cuando las quieras tomar solo abres el pastillero y tomas las 3 pastillas, eso mismo hace una funcion, hacer el proceso una sola vez y cada que quieras realizar la accion solo mandas a llamar la funcion que seria el pastillero y ejecutas la accion que viene en la funcion, lo mismo que tomar las pastillas dentro del pastillero
Declaraciones if = todo el tiempo estamos tomando decisiones y si tomas una decision dejas de tomar otra, entonces si tienes hambre que haces? comes y si no comes que pasa? te quedas con hambre, listo cada que quieras tomar una decision cambia el si por el if y "si no" por un else y asi lo entiende igual la computadora
Valores booleanos (true, false)= quieres hacerle saber a la computadora que algo es verdad para ti, como papa noel, si para ti es verdad tienes que hacerselo saber a la computadora, entonces le  dices que papa noel es True o verdad para ti, pero si eres un chico sin fe entonces para ti no existe papa noel entonces igual le tienes que decir a la computadora que para ti papa noel es un chiste entonces le pones papa noel es False, es decir le haras saber a la computadora a traves de esta palabra que se llama boolean para la computadora asi la computadora lo conoce como boolean, recuerda que quieres habalr el idioma de la computadora entonces te tienes que acostumbrar a sus expresiones.
